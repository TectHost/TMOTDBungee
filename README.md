# TMOTD Bungee
Simple optimized motd plugin

# Features
- MOTD
- Multiple motds
- Customizable change interval time
- Reload command
- MOTD set comand
- Fake players

# Commands
- /motd set (line 1) (line2)
- /motd reload
- /motd version

# Permissions
- tmotd.reload
- tmotd.set
- tmotd.version

# Dependencies
- None
